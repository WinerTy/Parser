# запуск проека


1. Клонирование репозитория 
    ```bash
     git clone --depth=1 https://github.com/WinerTy/Parser.git
     ```

## Bash Script
Установить python и pip 
```bash
sudo apt install python3 python3-pip -y
```

Выполнить команду в терминале
```bash
bash run.sh
```

Для запуска в фоновом режиме
```bash
bash run.sh &
```
