# запуск проека


1. Клонирование репозитория 
    ```bash
     git clone --depth=1 https://github.com/WinerTy/Parser.git
     ```

## Docker
Создание сборки
```bash 
    docker build -t tg_bot .
```
Запуск контейнера 
```bash
    docker run tg_bot 
```


## Bash Script
Установить python и pip 
```bash
sudo apt install python3 python3-pip -y
```

Выполнить команду в терминале
```bash
sh run.sh
```
