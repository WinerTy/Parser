# запуск проека


1. Клонирование репозитория 
    ```bash
     git clone --depth=1 https://github.com/WinerTy/Parser.git
     ```

## Docker
Создание сборки
```bash 
    docker build -t tg_bot .
```
Запуск контейнера 
```bash
    docker run tg_bot 
```


## Bash Script
Выполнить команду в терминале 
```bash
sh run.sh
```
